# Services

[![](https://jitpack.io/v/shanpark/Services.svg)](https://jitpack.io/#shanpark/Services)

## Install

To install the library add: 
 
   ```gradle
   repositories { 
       ...
       maven { url "https://jitpack.io" }
   }
   
   dependencies {
       implementation 'com.github.shanpark:services:0.0.4'
   }
   ```
